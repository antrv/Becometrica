# Becometrica

The repository contains various useful utilities.

### Becometrica.Unsafe
The project aggregates all unsafe code and implements types extremely useful for interop with the native code:   
- Pointers and constant pointers to an unmanaged structure
- Pointers to a UTF-8 string
- Conversion utilities

### Becometrica.Math
- Int128 and UInt128 types

### Becometrica.Math.Multiprecision
- Interop to mpir library
- Multiprecision numeric types: MpInteger, MpRational, MpFloat

### Becometrica.Parsing
Parser combinators library.
